# AZIZVERSE — Portfolio Website

A modern, elegant, high-end portfolio website built with Next.js 14, Tailwind CSS, and Framer Motion.

## ✨ Features

- Apple/Linear-inspired minimal design
- Framer Motion smooth animations
- Fully responsive (mobile-first)
- Contact form with success state
- Sticky blur navbar with scroll detection
- Inter typography with clean hierarchy

## 🛠️ Tech Stack

- **Next.js 14** (App Router)
- **Tailwind CSS v3**
- **Framer Motion v11**
- **TypeScript**

---

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/azizverse-portfolio.git
cd azizverse-portfolio
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run development server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### 4. Build for production

```bash
npm run build
npm run start
```

---

## 🌐 Deploy to Vercel

### Option A — Vercel CLI

```bash
npm install -g vercel
vercel
```

Follow the prompts. Your site will be live at `https://your-project.vercel.app`.

### Option B — Vercel Dashboard

1. Go to [vercel.com](https://vercel.com) and log in
2. Click **"Add New Project"**
3. Import your GitHub repository
4. Keep all default settings (Next.js is auto-detected)
5. Click **"Deploy"**

Your site is live! Every `git push` to `main` triggers automatic redeploy.
